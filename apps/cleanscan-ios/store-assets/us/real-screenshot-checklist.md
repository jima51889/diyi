# Real Screenshot Checklist

Capture screenshots from the real iPhone build. Do not submit mockups that show features differently from the app.

## Device

Use a 6.7-inch iPhone screenshot set if possible.

## Screens to Capture

1. Home screen with Document / Receipt mode visible
2. Apple document scanner screen while scanning paper
3. Review screen with multiple pages
4. Document detail with PDF tools
5. Receipt detail with merchant, date, and total fields
6. Recognized text screen
7. Signature capture screen
8. PDF preview / share screen
9. Settings screen showing privacy and support links

## Rules

- Use real app screens.
- Use English UI.
- Use non-sensitive sample documents.
- Do not show personal IDs, real receipts, addresses, or phone numbers.
- Do not show paid features for the first free release.
- Do not mention "AI" in screenshot captions.

## Sample Documents

- Fake receipt from a made-up store
- Simple printed note
- Sample contract page with placeholder text
- Classroom-style notes with no personal information
